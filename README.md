# IMDB Scraper 

## About 

A simple IMDB Scraper from taking [Learn Web Scraping with Node.js - Crash Course](https://www.udemy.com/course/nodejs-web-scraping/)

### Features 

* Movie Title
* Rating 
* Release Date 
* Popularity 
* Total Rating 
* Export to .csv

## Set up

### Clone

```bash
git clone
```

### Install 

```bash
npm install
```
